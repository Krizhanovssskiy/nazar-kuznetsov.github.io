vanilla-slider написан на чистом JavaScript без использывание библеотеки jQuery тоесть для его работы не нужно подключать jQuery
-------------------------------------------------------------------------------
Что может vanilla-slider

1. бесконечная прокрутка с анимацией прокрутка по горизонтали и вертикали
2. прокрутка с помощью мишкой
3. прокрутка с помощью свайпа
4. адаптивная высота
5. прокрутка в режиме fadeIn
6. авто прокрутка
7. добавляет класс active слайду и dots
8. в момент прокрутки слайдеру добавляеться класс animate и animate-Y там и меняем анимации
9. добавил css style файл
11. оптимизировал код
12. наслаждаемся!
----------------------------------------------------------------------------------

 				ИНСТРУКЦИЯ
1. Подключить стили vanilla-slider.min.css
2. Подключить script vanilla-slider.min.js перед закрывающим тегом </body>
3. написать разметку слайдера div.slider-container>ul.slider>li*5.slider-item
4. иницализация слайдера

	new Slider({
		container: '#slider',  //имя селектора в котором прямые элементы это слайды
	});

4. за скорость по прокрутки по горизонтали и вертикали отвечает стили с классом animate-X и animate-Y значение transition-duration должно быть больше 0 ms по умолчанию скорость будет 300ms
----------------------------------------------------------------------
	иницализация слайдера
	
	* означает обезательный параметр

	new Slider({
		container: '#slider',  * имя селектора пример #slider можно и класс
		next: '#next',         	 имя селектора кнопки врепед
		prev: '#prev',           имя селектора кнопки назад 
		dots: '.dots',           имя селектора dots это обычно точки или картинки *
		vertical: true, 				 вертикальный режим
		fade: true,						   в режиме fadeIn
		auto: true,              авто прокрутка (по умолчанию 4сек)
		time: 10000,             установить для авто прокрутки в милисекундах
	});
